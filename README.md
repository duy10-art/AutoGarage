# GarageOOP

## Beschreibung
Dieses Projekt wurde im Rahmen der Projektarbeit im Modul OOP mit IntelliJ IDEA CE erstellt
Es ist ein Java-Programm mit grafischer Benutzeroberfläche auf Basis von Swing
Mit dem Programm können Autos für eine Garage angelegt und in einer Liste verwaltet werden
Zusätzlich können die angelegten Autos gefiltert, ausgewertet und entfernt werden

##Projektstruktur
Das Projekt is in 4 Packages aufgeteilt

app
- Main - Main methode zum Starten des Programms

model 
- Auto - Modellklasse zur Beschreibung eines Autos der Garage mit Attributen und Methoden

gui
- GarageGui - Grafische Benutzeroberfläche und Verwaltung der Fahrzeugliste

test
- Autotest - JUnit-Test zur Überprüfung von Methoden der Klasse Auto

##Funktionen
- Anlegen neuer Autos über die Benutzeroberfläche
- Speichern der Autos in der Liste
- Anzeige aller Autos in einer Tabelle
- Filtern von Oldtimern
- Berechnung des gesamten Garagenwerts
- Sortieren der Tabelle nach einzelnen Spalten

Die gespeicherten Daten sind nur während der Laufzeit des Programms verfügbar

##Nutzung
Das Programm wird über die Main Klasse gestartet
Die Fahrzeugdaten werden über die Eingabefelder eingegeben und mit dem Speicher-Button/Enter 
zur Liste hinzugefügt  
Weitere Buttons ermöglichen Filterungen und Auswertung der gespeicherten Autos
